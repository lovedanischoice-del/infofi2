# 클라우드플레어 Pages 직접 배포 가이드

## 방법 1: Wrangler CLI 사용

1. Wrangler 설치:
```bash
npm install -g wrangler
```

2. 로그인:
```bash
wrangler login
```

3. 배포:
```bash
wrangler pages deploy dist --project-name=infofi2
```

## 방법 2: 드래그 앤 드롭 업로드

1. Cloudflare Pages 대시보드 접속: https://dash.cloudflare.com/
2. Pages 섹션으로 이동
3. "Upload assets" 클릭
4. `dist` 폴더 전체를 드래그 앤 드롭
5. 프로젝트 이름 입력: infofi2
6. "Deploy site" 클릭

## 방법 3: Git 연결 (자동 배포)

1. GitHub에 푸시 (인증 문제 해결 후):
```bash
git push origin main
```

2. Cloudflare Pages에서 GitHub 저장소 연결:
   - Framework preset: Vite
   - Build command: npm run build
   - Build output directory: dist
   - Node.js version: 18

3. 자동 배포 시작됨

---

## 배포 후 확인

배포가 완료되면 다음 URL에서 확인 가능:
- https://infofi2.pages.dev (또는 할당된 URL)

브라우저 캐시를 삭제하고 새로고침하세요 (Ctrl + Shift + R)
